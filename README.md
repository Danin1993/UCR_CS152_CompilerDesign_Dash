# Language Features

| Language Feature | Code Example |
|----------|----------|
| Add | x+y |
| Sub | x-y |
| Multiply | x*y |
| Divide | x/y |

| Less Than | x<y |

| Equality | x===y |

| Greater Than | x>y |
| Greater Than Equal | x>=y|
| Write | prt(x) |
| Read | read(x) |
| Arrays | int [8] array |
